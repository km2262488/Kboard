# Keyboard - Termux Versi Clean

Keyboard custom Termux versi clean buat dev & daily use. Semua tombol kepake, nggak ada yang mubazir.

### **Layout v1.8**
[git clone][&&][ALT][HOME][END][DEL]
[UP][LEFT][RIGHT][pip install]
[cd ][pkg ][nano ][ENTER][ls][python][clear]

### **Penjelasan Tombol**
| Tombol | Fungsi |
| --- | --- |
| `git clone ` | Clone repo 1 tap, udah ada spasi |
| `&&` | Gabungin command biar satset |
| `pip install ` | Install package Python cepet |
| `python ` | Jalanin script.py langsung |
| `nano ` | Buka text editor tanpa ngetik |
| `ls` | List file simple |
| `cd ` | Pindah folder, udah ada spasi |

### **Cara Install**
```bash
pkg update && pkg install python -y
git clone https://github.com/km2262488/Kboard.git
cd Kboard
python key.py
exit
> exit dulu lalu buka ulang Termux biar keyboard barunya aktif.### *Catatan Penting*
